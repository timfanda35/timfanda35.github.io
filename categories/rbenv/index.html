MAC OSX ruby console 無法輸入中文

2015.10.13
ruby rbenv shell

我是在 MAC OSX 上使用 rbenv 安裝 ruby，在換了新電腦重新安裝環境後突然發現在不管在 irb 還是 pry 中都無法輸入中文，而且操作上怪怪的。

在網上搜尋後找到了這篇 mac 无法在irb中输入中文

首先用 Homebrew 安裝 readline

brew install readline
