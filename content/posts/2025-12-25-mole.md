## 簡介

Mole 是一個用於清理硬碟空間的指令列工具，可以快速找出並刪除不需要的檔案。

GitHub Repo: [tw93/Mole](https://github.com/tw93/Mole)

## 安裝

Linux

```bash
curl -fsSL https://raw.githubusercontent.com/tw93/mole/main/install.sh | bash
```

MacOS 可以使用 Homebrew 安裝

```bash
brew install tw93/tap/mole
```

## 使用方法

Mole 提供了多種功能，以下是幾個我本身常用的指令。

### 刪除 Mac App 殘留檔案

```bash
mo uninstall
```

完整刪除 Mac App 與其相關的檔案。一般如我們將 Mac App 拖到垃圾桶刪除，會遺留一些設定檔案在系統中，使用 Mole 可以幫我們找出並清理這些殘留檔案。

{{< figure src="/images/2025-12-25/mole-uninstall.png" alt="mo uninstall" caption="https://github.com/tw93/Mole" >}}

### 硬碟空間分析

```bash
mo analyze
```

使用 Mole 來分析硬碟空間使用情況，找出佔用大量空間的檔案和資料夾。

再依據需求使用**空白鍵**選取要刪除的項目，最後按下**刪除鍵**來刪除選取的檔案。

{{< figure src="/images/2025-12-25/mole-analyze.png" alt="mo analyze" caption="https://github.com/tw93/Mole" >}}


### 清理軟體開發套件

```bash
mo purge
```

這個指令會找出如 node_modules、.venv 等開發環境相關的資料夾，讓我們選擇是否要刪除以釋放空間。

{{< figure src="/images/2025-12-25/mole-purge.png" alt="mo purge" caption="https://github.com/tw93/Mole" >}}
