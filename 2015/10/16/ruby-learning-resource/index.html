有任何錯誤或建議都歡迎告知 :)

Ruby ¶

Learn Ruby The Hard Way - 笨方法學 Ruby ¶

練習量很足夠的電子書，推薦用來練基本功。 另外可以先看這篇 閱讀 Learn Ruby The Hard Way 時可能會遇到的問題 做一下心理建設 XD

Code Cademy ¶

練習量很充足的互動課程，完成關卡得到的 Badge 更能增加成就感。

Code School ¶

Try Ruby 很適合入門體驗 Ruby 寫作的感覺。有許多主題式的課程與影片，大多數都需要付費才能觀看。完成課程的 Badge 可以貼到 Linkedin 上。

Ruby Document ¶

Ruby 的文件，有空就翻一翻，說不定會有新的體悟。

Ruby Warrior ¶

用 Ruby 操作主角破關的遊戲。類似的還有CodeCombat

Codewars ¶

有很多題目關卡可以讓你練習，不過也有很多需要知道演算法才有辦法過關。 你也可以閱覽其他人的程式碼，對於寫作大有幫助。

The odin project ¶

w3cshool.cc ¶

Rails ¶

Rails Girl ¶

友善的入門網站，不只提供教材，也有教導如何舉辦工作坊的指南。如果你想帶你的朋友入門卻不知從何下手，來看看這個網站吧。(可以關注Railsgirls Taiwan以獲得最新活動訊息)。

RailsBridge ¶

RailsBridge 工作坊教學文件，除了教材也有教導如何舉辦工作坊的指南。

Rails 101 ¶

令新手不斷思考的教材，多數問題的解答都在留言中。

Rails 101s ¶

由 Rails 101 擴充的教學，有較多的說明內容。

Rails 102 ¶

Xdite 對於使用 Rails 開發網站的經驗分享。

Ruby on Rails 實戰聖經使用 Rails 4.2 及 Ruby 2.2 ¶

這應該是最知名的 Rails 中文工具電子書了XD，對於 Rails 有著很詳盡的解說。想要深入了解如何使用 Rails，可以先從這個網站開始，再去嗑 Rails Guide。

Rails Tutorial 3rd ¶

聽說是網路上最推薦的原文電子書，有能力的話就贊助一下作者吧！

How I finally learned to build stuff with Rails ¶
