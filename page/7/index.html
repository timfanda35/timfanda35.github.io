用 Ruby 建立 Web 專案的一些起始 gem

2015.10.17
ruby gem

Ruby 有許多 gem 可以用來做 web 開發，以下列出一些我所找到可以作為起頭的 gem 提供大家參考。

有任何錯誤與建議歡迎告知:)

Web

Rack

提供 webserver 最小的功能介面，大多 Ruby 的 web 開發框架都是基於 Rack 所開發的。

Rails

最知名的魔法書，能快速地開發功能豐富的網站。

Sinatra

比 Rails 輕量的開發框架，適合用來開發小型專案或是 API。

Padrino

基於 Sinatra 的開發框架，提供更多方便的資源開發網站。

Cuba

試圖比使用 Sinatra 寫更少 code 的開發框架，官方 Slide 表示效能比 Sinatra 好。

Franky

產生小型的 web 專案，適合用來實驗想法的小專案。

Ruby 的 %Q, %q, %W, %w, %x, %r, %s, %i

2015.10.14
ruby

有時候在看其他人的 Ruby 程式碼時，常常發現有 %w, %r 等 % 開頭的程式碼。上網查了一下，才知道這些叫做 Percent literals(或Percent Strings)。

以下內容是改寫自 %Q, %q, %W, %w, %x, %r, %s 與 Ruby 里的 %Q, %q, %W, %w, %x, %r, %s, %i (译)。

執行環境：

  • ruby: 2.2.2
  • console: irb

%Q

代替雙引號字串，如果在字串中有很多雙引號的時候很好用

>> what_frank_said = 'Hello!'
=> "Hello!"

>> %Q(Joe said: "Frank said: "#{what_frank_said}"")
=> "Joe said: \"Frank said: \"Hello!\"\""

>> puts %Q(Joe said: "Frank said: "#{what_frank_said}"")
Joe said: "Frank said: "Hello!""
=> nil
