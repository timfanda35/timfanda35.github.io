我想要推薦這一個網站 InstallRails.com

這個網站一步步教你在不同平台安裝開發 rails 的環境，圖文並茂，說明詳細

以 Mac 平台來說，會安裝的項目有：

  • Xcode
  • Homebrew
  • Git
  • RVM
  • Ruby
  • Rails
  • Sublime Text

不過在教學中安裝管理 Ruby 版本的工具是 rvm

我自己喜歡用的管理 Ruby 版本的工具是 rbenv

除了安裝 Rails，在請別人安裝 Ruby 的時候我也會丟這個網站給他，請他依步驟做到安裝 RVM
